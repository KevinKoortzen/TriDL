# 🚀 TriDL

A seamless, all-in-one platform for downloading videos from TikTok, YouTube, and Instagram with speed and simplicity.

---

## 📥 Download

Get the latest version here:

👉 **[Download TriDL](https://github.com/KevinKoortzen/TriDL/releases/latest)**

> No setup required — just download and run.

---

## 📌 Features

* 🎥 Download videos from **TikTok**
* ▶️ Download videos from **YouTube**
* 📸 Download videos from **Instagram**
* ⚡ Fast and easy link detection
* 🧠 Automatically identifies the platform from the URL
* 💻 Simple and user-friendly interface

---

## 🛠️ How It Works

1. Copy the link of the video you want to download
2. Paste it into TriFetch
3. The app detects the platform automatically
4. Click download — done ✅
